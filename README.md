# onlineExam
本系统在实现过程中主要采用了Spring Boot框架、应用服务器Tomcat9.0、数据库服务器MySQL8.0.20、数据库持久层框架MyBatis、Java模板引擎Thymeleaf、前端框架Bootstrap、数据库连接池Druid、jQuery、AJAX、弹出层组件Layer、lombok注释Java库。
